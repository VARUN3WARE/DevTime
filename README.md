# DevOS

A local-first developer productivity tracker that understands your workflow.

## Features
- Background tracking of file and project activity
- Automated git project detection
- Idle time detection
- Rich terminal dashboard
- Flow score and productivity insights
- Replay timeline of your work

## Installation
```bash
pip install -r requirements.txt
pip install -e .
```

## Usage
- `devos start`: Start the tracking daemon
- `devos stop`: Stop the tracking daemon
- `devos status`: Check daemon status
- `devos today`: See today's productivity stats
- `devos insights`: Get AI-powered workflow insights
- `devos replay`: Replay your work timeline
